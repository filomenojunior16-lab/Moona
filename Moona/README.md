# Moona - Feliz Aniversário

Página web de saudação e presente digital para Marilca Basílio.

## Como usar

- Abra `index.html` no navegador para testar localmente.
- A página já está preparada para visualização em smartphones Apple.
- O layout responde em telas pequenas e mantém a experiência consistente.

## Hospedagem no GitHub Pages

1. Inicialize um repositório Git na pasta do projeto:
   ```powershell
   git init
   git add .
   git commit -m "Inicial commit do site de aniversário"
   ```
2. Crie um repositório no GitHub e adicione o remoto:
   ```powershell
   git remote add origin https://github.com/SEU_USUARIO/SEU_REPOSITORIO.git
   git push -u origin main
   ```
3. No GitHub, ative o GitHub Pages nas configurações do repositório.
   - Selecione a branch `main` e a pasta `/ (root)`.

## Estrutura

- `index.html` — página principal
- `css/style.css` — estilos responsivos
- `js/script.js` — lógica de validação e confetti
- `assets/marilca.jpg` — foto utilizada no presente digital
